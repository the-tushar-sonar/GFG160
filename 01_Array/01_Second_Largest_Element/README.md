# Second Largest Element

## Problem
Find the second largest element in an array.

## Approaches

### Brute Force
- Sort the array
- Time Complexity: O(n log n)
- Space Complexity: O(1)

### Better
- Two passes
- Time Complexity: O(n)

### Optimal
- Single traversal
- Time Complexity: O(n)
- Space Complexity: O(1)

## Learning
- Learned how to track largest and second largest simultaneously.
- Important edge case: duplicate maximum values.
